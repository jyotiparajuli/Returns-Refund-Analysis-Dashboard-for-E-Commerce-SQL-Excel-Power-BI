# Returns-Refund-Analysis-Dashboard-for-E-Commerce-SQL-Excel-Power-BI

##  Project Overview

This project demonstrates an end-to-end data analytics workflow using **SQL, Microsoft Excel, and Power BI**. It focuses on analyzing e-commerce returns and refunds, uncovering customer and seller insights, identifying business challenges, and providing data-driven recommendations through an interactive dashboard.

##  Demo & Project Files
https://drive.google.com/drive/folders/1RqBvOFysMwNRxp0WacD-vYhfYpiLRjUB?usp=sharing

##  Project Objectives

* Analyze revenue, orders, returns, and return rate.
* Identify top-performing products and categories.
* Understand return reasons and payment trends.
* Explore customer and seller insights.
* Generate actionable business recommendations.

##  Tech Stack

* **SQL** – Data querying and business analysis
* **Microsoft Excel** – Data cleaning and preprocessing
* **Power BI** – Data modeling, DAX measures, KPIs, interactive dashboards, and business storytelling
